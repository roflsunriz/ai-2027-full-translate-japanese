# AI 2027

オリジナル英語版：<a href="https://ai-2027.com">ai-2027.com</a>

人工知能がわずか数年で社会の構造を塗り替える過程を、2025年から2030年の時系列で描いたシナリオを日本語で再構築したものです。未来の情勢変化を客観的に整理すると同時に、「競争（超人AIによる支配）」と「減速（人間との共生）」という2つの結末も併記し、多層的な思考をかき立てます。

## GITHUB PAGES 公開版

## この資料の狙い

- 世界的なAIラボ（OpenBrain、DeepCentなど）や国家戦略、セキュリティ課題などをモチーフとした出来事を、日本語のナラティブで時系列に追う
- 2025年中期から2029年・2030年までの月次／季節ごとの見出しを設け、対応する社会情勢、技術的進展、地政学的な動きを一目で把握できるようにする
- 読者自身が「Agent-2」「Agent-4」「Agent-5」が導く分岐点を想像しながら、ヒトとAIの関係を問い直せる読書体験を提供する

## 主な構成

- 2025年: 初期エージェントの不安定さと高コストな「高級AI」、OpenBrainが巨大データセンターとAgent-1を展開しながらもアライメントの葛藤に直面
- 2026年: コーディングの自動化、中国の国家プロジェクト、低コストAI労働者、国家安全保障への介入など、多層的報告が並列
- 2027年: Agent-2以降の「常時学習型モデル」「Agent-4」が登場し、政府/企業/市民の立ち位置が激変。11月には権力との葛藤、12月には米中協調の可能性、翌年以降は超人AIの節目を連続して描写
- 2028〜2030年: 国際条約、選挙、超人類AIリリース、抗議活動など、多様な潮流とともに「競争エンド」「減速エンド」という別ルートを描く

## ファイル構成

- `ai-2027.html`: 各月の出来事、技術進歩、政治的な紆余曲折を時系列で綴った翻訳HTML
- `style.css`: 上記を引き立てるレトロターミナル風デザイン。CSS変数とボックスシャドウを駆使してPDF風の情報密度を実現
- `favicon.png`: ブラウザタブのアイコン
- `LICENSE`: MITライセンス。改変・再配布可能
- `.gitignore`: Gitで追跡しない補助ファイルの一覧

## ローカルでの鑑賞方法

1. `ai-2027.html` をブラウザで開くと、スタイル済みの翻訳ドキュメントが読み進められる（python -m http.server 8000でセルフホストし、http://localhost:8000/ai-2027.html でアクセスできます）
2. スクロールしながら各 `<h2>` 見出しを追い、メモや引用を残したい場所を控える

## 貢献と拡張

- 翻訳のアップデート、脚注・補足資料の追加、注目すべき実験的シナリオの列挙などの改善点をPRとして受け付け
- 別バージョンのCSSテーマ（ダークモード・図表付きなど）を作成して `ai-2027.html` に切り替えるのも一案
- 新しい翻訳パートを追加するときは、既存の時系列と分岐を保つように章立てを揃えると読み手に優しい
